```markdown
# Pourō v0.1.0 — Public MVP

## Public URL

https://ns-del346.github.io/Pouro-GPT/

## What is included

- Mobile-first pour-over brewing guide PWA
- Brew method selection
- Recipe setup
- Brew timer
- Brew finish note
- Brew history
- History detail
- Rebrew
- Settings
- In-app About / Sources / Legal / Privacy pages
- GitHub Pages deployment
- GitHub Pages SPA fallback for direct nested-route access
- Public README / NOTICE / PRIVACY documentation

## Included methods

- 4:6 Method
- Hybrid
- 10 Pour Method
- Ice Brew

`10 Pour Method` is the MVP user-facing method label. `THE NEO BREW` is not
promoted as the Brew Home method name in this release.

## Verification status

- Public root URL loads.
- Settings / About / Sources / Legal / Privacy pages load.
- Direct nested routes avoid the GitHub Pages branded 404.
- `375 x 667` smoke QA passed.
- Build passed.
- No blocking public-release issues recorded.

## Notice

- Pourō is unofficial.
- Pourō is not affiliated with, sponsored, endorsed, supervised, or approved
  by 粕谷哲氏, PHILOCOFFEA, HARIO, or any referenced person, company, maker, or
  brand.
- Third-party names and method names are used for descriptive reference only.
- Method data may include verified, researched, summarized, interpreted,
  app-guided, unverified, or placeholder values depending on metadata.
- Users should consult original sources where accuracy or faithful
  reproduction matters.

## Privacy / storage

- Brew history and settings are stored in browser `localStorage`.
- No user account.
- No Pourō backend.
- No Pourō cloud sync.
- No dedicated in-app analytics tracking.
- Clearing browser or site data may delete saved history and settings.

## Known limitations

- Some method data may still be placeholder or needs review.
- Separate Preview screen is not included in v0.1.0.
- PWA / offline behavior may depend on browser cache and Service Worker state.
- No account system.
- No cloud sync.
- No TDS / Extraction Yield.
- No Bluetooth scale integration.
- No community features.
- No analytics dashboard.

## Non-goals

- Not an official recipe reproduction app.
- Not a complete method encyclopedia.
- Not a medical or health advice tool.
- Not a cloud account product.
- Not a social/community app.
```
