## 7. Data, source, and recipe status

- Depending on metadata, method data may include placeholder, needs review,
  third-party, researched, interpreted, app-guided, verified, or official-like
  statuses.
- Metadata such as `sourceStatus`, `verificationLevel`, and
  `valuesArePlaceholder` exists to avoid overclaiming.
- An official-like source/status description does not establish affiliation,
  endorsement, supervision, sponsorship, or approval.
- Users should not treat unverified or placeholder values as official original
  recipes.
- Users should consult original sources where accuracy or faithful reproduction
  matters.

## 8. Privacy and storage

- Brew history and settings use browser `localStorage` only.
- No user account.
- No Pourō backend.
- No Pourō cloud sync.
- No dedicated in-app analytics tracking.
- Clearing browser or site data may delete saved history and settings.
