## Summary

Pourō-GPT Ver1.0 is a mobile-first brewing execution guide and timer PWA. It
supports the core flow from method selection and recipe setup through the brew
timer, finish, local history, and rebrew.

## Included in Ver1.0

- Brew Home and method selection
- Recipe Setup
- Brew Timer with target-total-first hierarchy
- Finish save or discard
- History list
- History Detail
- Rebrew
- Last Brew shortcut
- Settings
- CSV export
- JSON backup export
- Delete history
- PWA manifest and icon path fix for GitHub Pages
- Service Worker and offline app-shell QA
- Legacy or corrupt `localStorage` guard
- Release candidate QA

## Supported Method Groups

- 4:6 Method
- Hybrid / HARIO Switch-style
- THE NEO BREW / 10 Pour
- Ice Brew

## Data and Privacy Model

Pourō-GPT has no account, backend server, or cloud sync. Brew history and
settings are stored in browser `localStorage`. CSV and JSON exports are created
locally as downloads. Ver1.0 does not include an import or restore UI.

## PWA / Offline Notes

The app is a static PWA served through GitHub Pages. The app shell can load
offline after the Service Worker has cached it. Offline behavior depends on
prior Service Worker and cache state and is not guaranteed.

## Source and Legal Notes

Pourō-GPT is personal and unofficial. Method names are descriptive
user-facing categories. Method data may be verified, researched, summarized,
interpreted, app-guided, unverified, or placeholder. It is not guaranteed to
be official or complete, and brew results are not guaranteed. See
[NOTICE.md](../../NOTICE.md) for the canonical source and legal caution.
