# Pouro-GPT

Pourō is a quiet pour-over coffee brew guide and timer PWA.

## Concept

Pour slowly. Brew deeply.

Pourō helps users follow source-based pour-over coffee recipes with clear brew steps, timer guidance, pour amount, cumulative water amount, and reflective brew history.

## Current Status

MVP planning and implementation preparation.

## Tech Stack

- React
- TypeScript
- Vite
- PWA
- Local storage
- No login
- No external API

## MVP Scope

- Brew Home
- Recipe Setup
- Brew Timer
- Brew Finish
- History
- History Detail
- Settings

## Non-Goals

- SNS
- Recipe posting
- Bean inventory
- Water management
- Roasting log
- Bluetooth scale integration
- Account system
- Subscription
- AI taste diagnosis
